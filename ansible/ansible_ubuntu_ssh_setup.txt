Ansible Installation and SSH Setup on Ubuntu

---

## 1. Update System
sudo apt update
sudo apt upgrade -y

## 2. Install Ansible
sudo apt install -y software-properties-common
sudo add-apt-repository --yes --update ppa:ansible/ansible
sudo apt install -y ansible
ansible --version

## 3. Copy client private key to Control Machine
Assuming you have `clientkey.pem` downloaded locally, move it to your control machine.

mkdir -p ~/exercises/exercise1
mv ~/Downloads/clientkey.pem ~/exercises/exercise1/
chmod 400 ~/exercises/exercise1/clientkey.pem

## 4. Create Ansible Inventory
Create a file `inventory` inside `~/exercises/exercise1/`:

all:
  hosts:
    web01:
      ansible_host: CLIENT_IP
      ansible_user: ubuntu
      ansible_ssh_private_key_file: /home/ubuntu/exercises/exercise1/clientkey.pem

Replace `CLIENT_IP` with the IP of your client node.

## 5. Test SSH via Ansible
From control machine, run:

ansible web01 -m ping -i ~/exercises/exercise1/inventory

Expected output:
web01 | SUCCESS => { "changed": false, "ping": "pong" }

## Notes
- Make sure `clientkey.pem` corresponds to the client node.
- Do not generate new SSH keys, just use the existing `clientkey.pem`.
- You can now run any Ansible commands or playbooks on `web01` using this inventory and key.
