Minikube Installation Steps on Windows
--------------------------------------

Prerequisites:
✔ Windows 10 or later with virtualization enabled.
✔ Install WSL2 or use Hyper-V or Docker as driver.

Step 1 – Install Dependencies
1. Install WSL2:
   Open PowerShell as Administrator and run:
   wsl --install
   Or follow: https://learn.microsoft.com/en-us/windows/wsl/install

2. Install kubectl:
   Option 1 – Download from https://kubernetes.io/docs/tasks/tools/install-kubectl-windows/
   Option 2 – Using Chocolatey:
   choco install kubernetes-cli

3. Install Minikube:
   Option 1 – Download from https://minikube.sigs.k8s.io/docs/start/
   Option 2 – Using Chocolatey:
   choco install minikube

4. (Optional) Install Docker Desktop:
   Download from https://www.docker.com/products/docker-desktop/

Step 2 – Start Minikube
Use one of the following commands depending on your driver:

For Docker driver:
minikube start --driver=docker

For Hyper-V driver:
minikube start --driver=hyperv --hyperv-virtual-switch="Primary Virtual Switch"

For VirtualBox driver (with WSL2):
minikube start --driver=virtualbox

Step 3 – Verify the Cluster
1. Check cluster status:
   minikube status

2. Check nodes:
   kubectl get nodes

3. Deploy a test application:
   kubectl create deployment hello-minikube --image=k8s.gcr.io/echoserver:1.4
   kubectl expose deployment hello-minikube --type=NodePort --port=8080

4. View services:
   kubectl get services

5. Access the test app in the browser:
   minikube service hello-minikube

Step 4 – Stop or Delete Minikube
Stop Minikube:
minikube stop

Delete Minikube:
minikube delete
